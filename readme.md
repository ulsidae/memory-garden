# 🌱 Memory Garden

**AI-powered web application for memory journaling and family-based reminiscence**

---

## 📌 Project Overview

Memory Garden is a **web-based memory journaling application (MVP)** that allows users to store, revisit, and organize personal memories using photos, videos, and text.

Users can upload media files and attach contextual descriptions such as time, place, and personal stories.  
An AI-assisted guide (planned) helps users structure these memories into more organized and readable entries.

This project is developed as a **startup idea competition prototype**, focused on:
- AI-assisted journaling
- Digital healthcare exploration
- Family-based memory sharing

---

## 🎯 Problem Statement

With an aging population, cognitive decline and dementia are becoming increasingly common.

Existing tools mostly provide:
- Static photo/video storage
- Simple album-based organization
- Limited interaction with stored memories

These approaches do not support:
- Active memory recall
- Structured storytelling
- Cognitive engagement through reflection
- Collaborative family input

Memory Garden explores a more interactive approach to memory storage through guided recall and journaling.

## 💡 Key Features (MVP)

### 📷 Memory Upload
- Upload images and videos via web interface
- Attach basic metadata:
  - Title
  - Time
  - Location (optional)
  - Description

### 📝 Memory Journal
- Text-based memory entries
- Simple tagging system for organization

### 🤖 AI Memory Guide (Planned)
- LLM-based conversational assistant (OpenAI / Google API TBD)
- Helps users recall and describe memories step-by-step
- Generates structured summaries from user input

### 👨‍👩‍👧 Family Sharing
- Shared memory pages per user/group
- Family members can view and add context

---

## 🧠 AI Interaction Design (Planned)

The AI system is designed as a **structured memory assistant**.

Core behavior:
- Ask one question at a time
- Use simple and clear language
- Maintain conversation context
- Summarize user input into structured memory data

### Example Flow:
1. Where was this photo taken?
2. Who is in this memory?
3. What happened at that time?

### Output:
- Structured memory entry (text + metadata)
- Short summary of the event
- Stored timeline-compatible record
